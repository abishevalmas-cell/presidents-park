# President's Park — Авторский надзор

Платформа отслеживания проекта отделки квартир 41–42–43 в ЖК President's Park.

## Деплой на Vercel

### 1. Создай репозиторий на GitHub
```bash
git init
git add .
git commit -m "President's Park tracker"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЮЗЕР/presidents-park.git
git push -u origin main
```

### 2. Подключи к Vercel
1. Зайди на [vercel.com](https://vercel.com) → **Add New Project**
2. Импортируй репозиторий `presidents-park`
3. Framework Preset: **Vite**
4. Нажми **Deploy**

Через ~30 секунд получишь ссылку вида `presidents-park.vercel.app`

## Стек
- React 18
- Vite 5
- Чистый CSS (без фреймворков)
